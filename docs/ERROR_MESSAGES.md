# Errori principali

## File non valido
Il file caricato non è un Excel `.xlsx` leggibile.

## Fogli mancanti
Manca uno o più fogli obbligatori del template.

## Vendite_2025 mancante o vuoto
Il motore si ferma perché il foglio principale non è disponibile.

## Output non generati correttamente
Il motore è partito ma non ha prodotto i file finali attesi.
